一键部署收件 Worker

点击上方「一键部署 Worker」按钮，将自动在你的 CF 账号下部署账号级收件 Worker， 并为该账号下所有域名配置 Email Routing catch-all → Worker， 域名下任意地址的邮件都会自动入库。

  • 每次部署会刷新所有域名签名密钥与 catch-all 路由
  • 新增域名后需重新部署一次以更新密钥映射
  • CF API Token 需具备 Account:Workers Scripts:Edit 权限
  • 每个域名独立签名密钥，验证 webhook 时按收件域名自动定位
