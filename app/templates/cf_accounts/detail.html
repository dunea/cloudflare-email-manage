一键部署收件 Worker

点击上方「一键部署 Worker」按钮，将自动在你的 CF 账号下部署账号级收件 Worker， 并为该账号下所有域名配置 Email Routing catch-all → Worker， 域名下任意地址的邮件都会自动入库。

  • Worker 名称：cf-email-manager-webhook（账号内唯一）
  • 为缺失密钥的域名自动补生成 webhook_secret，已有密钥沿用
  • 每次部署会刷新所有域名的 catch-all 路由
  • 新增域名后需重新部署一次以更新密钥映射
  • CF API Token 必须通过上方权限预检，尤其需要 Workers Scripts: Edit、Zone Settings: Edit 和 Email Routing Rules: Edit
  • 每个域名独立签名密钥，验证 webhook 时按收件域名自动定位
